find_filter(filter_cls)[source]¶

Finds an existing filter using a filter class filter_cls. If not found, None is returned.

This method is useful in cases where one wants to modify and extend and existing clause, a common example might be an And filter. The method only looks in the query’s top-level filter.

Param:filter_cls The the filter class to find.
